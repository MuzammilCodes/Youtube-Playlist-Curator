Privacy and Data Usage Policy

Your privacy is important. This website does not track, collect, store or share any personal data. The only interaction with user data occurs when the website integrates with the YouTube Data API. Here's how it works:

Data Source: The website retrieves data from the YouTube Data API once the user logs in with their Google account.
Actions on Data: Users can view and manage their YouTube playlists directly. Actions such as selecting or removing
videos are performed in real-time through the YouTube Data API.
No Data Storage: The website does not save any user data on its servers. All data interactions are strictly limited to the user's current session and occur between the user's browser and the YouTube Data API.
Transparency: The website operates under the principle of minimal data usage, solely for the purpose of enhancing user control over their YouTube playlists.

This website uses only essential session cookies necessary for Google OAuth authentication. No tracking or marketing cookies are used.
Data Protection Mechanisms:
  • Secure HTTPS Connections: All data transmissions are encrypted using HTTPS to protect user data during transit.
  • OAuth 2.0 Protocol: We use OAuth 2.0 for secure authentication.
  • Session-Only Data Handling: User data is handled only during the active session and is not stored on our servers.
  • Access Controls: Only authorized personnel have access to the systems that interact with the YouTube Data API.
Retention and Deletion of Google User Data:
  • No Data Storage: As we do not store any user data, there is no retention period. All data interactions are session-based and are terminated once the session ends.
  • User Data Deletion: Users can manage their data directly through the YouTube Data API. Since we do not store any data, there is no need for a separate deletion process.
This application uses Google APIs and adheres to the Google Terms of Service (https://policies.google.com/terms)
By ensuring that no data is tracked or stored, the website aligns with best practices for user privacy and protection.
